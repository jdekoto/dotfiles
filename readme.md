# dotfiles
A small little configuration for my PostmarketOS Chromebook.

**Apps**:
Compositor: Niri
Shell: Noctalia
Terminal: Alacritty (fish)
Monitor: Btop++
File Explorer: Nautilus/yazi
Editor: Neovim
Web: Epiphany
Candy: scope-tui, cmatrix, and cbonsai
Everything Else:
    Whisp
    Warehouse
    Flatseal
    Camera (snapshot)
    Gnome Sound Recorder
    Rewaita
    Gnome Software
    Gnome Disks
    Gnome Text Editor
    dconf Editor
    Gnome Tweaks
    Audio Player (decibels)
    Video Player (showtime)
    nwg-look
    Dissent
    Pico-8
    Picotron

**Notes**: Noctalia is on Version 5, so it's not the Quickshell based shell you'd expect. Everything else had been chosen to either improve and/or optimize the experience on a 2017 Celeron.
